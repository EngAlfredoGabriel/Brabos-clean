# 🚀 Brabos Clean

Sistema web desenvolvido para apresentação e agendamento de serviços de higienização profissional, com foco em praticidade, conversão de clientes e experiência visual moderna.

---

# Sobre o Projeto

O **Brabos Clean** foi criado com o objetivo de facilitar a conexão entre clientes e serviços de limpeza especializada, oferecendo uma interface intuitiva e otimizada para conversão direta via WhatsApp e agendamento rápido.

Ideal para negócios que desejam presença digital profissional e aumento de demanda.

---

# Principais Funcionalidades

 📅 Agendamento de serviços online
 📲 Redirecionamento direto para WhatsApp
 🎨 Interface moderna e responsiva
 ⚡ Carregamento rápido com Vite
 📱 Adaptado para dispositivos móveis

---

# Tecnologias Utilizadas

 **React**
 **Vite**
 **JavaScript**
 **CSS3**

---

# Como Executar o Projeto

Clone o repositório:

```bash
git clone https://github.com/EngAlfredoGabriel/brabos-clean.git
```

Acesse a pasta do projeto:

```bash
cd brabos-clean
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```
